# Piece art

The chess piece SVGs in this directory are the **Cburnett** set by
**Colin M. L. Burnett**, the widely-used Staunton piece art from online chess.

- Source: distributed with Lichess — https://github.com/lichess-org/lila
  (`public/piece/cburnett/`)
- License: GPLv2+ / GFDL / BSD (triple-licensed via Wikimedia Commons)

These assets keep their original license; the rest of this project is MIT.
To use a different set, replace `{w,b}{K,Q,R,B,N,P}.svg` here and update the
imports in `src/components/Piece.tsx`.

> chess.com's own "neo" piece set is proprietary and cannot be redistributed,
> so this open Staunton set is used to give the same polished, real-chess look.
